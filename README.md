# Student Enrollment Form using JsonPowerDB

This is a simple Student Enrollment Form developed as part of the JsonPowerDB V2.0 course project.

###  Features
- Add new student details  
- Fetch existing student information using Roll No  
- Update existing student details  
- Delete a student record  
- Simple and clean UI  
- Uses JsonPowerDB for serverless, fast CRUD operations  

###  Database Details
- **Database Name:** Student_db  
- **Relation Name:** Student_data  
- The database and relation are auto-created during the first insert operation.

###  Technologies Used
- HTML  
- CSS  
- JavaScript  
- jQuery (for AJAX)  
- JsonPowerDB API  

###  How It Works
1. Enter a new student’s details → Click **Save**  
2. To fetch student details → Enter Roll No → Click **Get**  
3. Edit and click **Update** to modify a record  
4. Click **Delete** to remove a record  
5. Click **Reset** to clear the form  

###  JsonPowerDB Advantages
- Fastest NoSQL JSON-based DBMS  
- No server setup required  
- Auto database & table creation  
- Ideal for rapid web application development
